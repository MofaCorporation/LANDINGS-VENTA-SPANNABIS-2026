# Los monstruos de Tarumba

Tienda **Tarumba's Farm**: cinco landings de coleccionables (ES/EN), checkout con **TPV Redsys** y stack PHP 8.2+ sin framework.

## Documentación

- `agents.md`, `project_config.md`, `workflow_state.md` — gobernanza para desarrollo con Cursor
- `estructura-proyecto.md`, `redsys-php-flujo.md`, `i18n-sistema-idiomas.md` — referencia técnica
- `context/` — mapas de rutas, Redsys, i18n, tokens de diseño, esquema SQL
- `EJEMPLOS/` — landings y checkout de referencia visual aprobados

## Estado

El código de aplicación (`public/`, `src/`, `templates/`) se implementa según `estructura-proyecto.md`. Este repositorio incluye gobernanza, docs y referencias de diseño.
